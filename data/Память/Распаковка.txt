# Распаковка (Unboxing)

## Как

Перенос данных из кучи в стек:

```
object obj = 1;

int i = (int)obj;
```

## Зачем

Выполнение применимых к значимому типу действий над упакованным в интерфейс или тип object экземпляром значимого типа.

## Почему

Если действие совершается над экземпляром значимого типа, перед выполнением этого действия над упакованным экземпляром этого же типа необходимо преобразовать этот экземпляр интерфейса или типа object к требуемому значимому типу.
