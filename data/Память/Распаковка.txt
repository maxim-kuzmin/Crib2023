# Распаковка (Unboxing)

## Как

Перенос данных из кучи в стек:

```
object obj = 1;

int i = (int)obj;
```

## Зачем

Выполнение действий, применимых к упакованным экземплярам конкретного значимого типа.

## Почему

Если действие совершается над экземпляром конкретного значимого типа, перед выполнением этого действия над упакованным экземпляром этого же типа необходимо преобразовать этот экземпляр интерфейса или типа object к требуемому значимому типу.
