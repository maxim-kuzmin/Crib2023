# Упаковка (Boxing)

## Как

Перенос данных из стека в кучу:

```
int i = 1;

object obj = i;
```

## Зачем

Выполнение действий, применимых к экземплярам любого значимого типа.

## Почему

Если действие должно совершается над экземпляром интерфейса или типа object, перед выполнением этого действия над экземпляром значимого типа необходимо преобразовать этот экземпляр к требуемому ссылочному типу.
